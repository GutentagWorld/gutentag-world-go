# gutentag-world-go

> Gutentag, World! in Go

## Run

```
go run cmd/gutentag-world/main.go
```


## Install

```
go get github.com/GutentagWorld/gutentag-world-go
```


### Usage

## Usage

```go
import gutentagworld "github.com/GutentagWorld/gutentag-world-go"

fmt.Println(gutentagworld.GutentagWorld())
```

Or from the command line:

```sh
go run ./cmd/gutentag-world
```
